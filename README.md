#easy-flex

> 能够快捷使用flex的css组件

```
flex-h横向布局，flex-v垂直布局，设置为主轴方向；
同时添加inline类名可以是元素变为inline的flex布局；
子元素排列方式使用视觉的上(top)下(bottom)左(left)右(right)作为class名；
子元素添加flex-fill的class名可以撑开，填充空间；
交叉轴默认居中，需要拉伸可以添加stretch类名；
如果有其它布局要求，请使用flex布局的官方属性来设置。
```
